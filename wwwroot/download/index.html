Après téléchargement

Les archives sont prêtes à l’emploi. Décompressez, lisez le README puis lancez l’exécutable correspondant à votre système.

Linux / macOS

Rendre le binaire exécutable si nécessaire puis lancer le jeu.

chmod +x ./lama ./lama

Windows

Lancez directement `lama.exe` depuis l’explorateur ou le terminal.

lama.exe
