Panorama technique

Le coeur metier vit dans les librairies .NET, avec une separation nette entre contrats, domaine, cas d'usage et infrastructure. L'interface CLI et le serveur online reutilisent les memes regles.

Architecture .NET

Couches principales: Lama.Contracts, Lama.Domain, Lama.Core, Lama.Infrastructure, avec adaptateurs Console et Server.

Contracts -> Domain -> Core -> Infrastructure

Gameplay et fiabilite

Moteur de jeu centralise, regles de pose, jokers et scoring; persistance JSON locale et mode online autoritaire.

CLI local + API online + tests unitaires

Plateforme de diffusion

Builds single-file pour Linux, Windows et macOS en x64/arm64, distribues en ZIP avec README.

https://playalama.online/download/
