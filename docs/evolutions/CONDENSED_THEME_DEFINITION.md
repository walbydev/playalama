# Nouveau thème condensé et suppression d'anciens thèmes
Ce thème condensé est conçu pour les utilisateurs qui préfèrent une interface plus compacte et minimaliste. 
Il réduit l'espace entre les éléments de l'interface, permettant ainsi d'afficher plus d'informations à l'écran sans avoir à faire défiler autant. 
Ce thème est idéal pour les utilisateurs qui travaillent avec de grandes quantités de données ou qui souhaitent maximiser l'utilisation de l'espace disponible sur leur écran.

## Suppression des anciens thèmes
Pour laisser de la place au nouveau thème condensé, nous avons décidé de cacher les anciens thèmes suivants:
- Verdâtre
- Bleuâtre
- Vermillon

## Définition du thème condensé
- Traductions :
  - Français : Compact 
  - Anglais : Condensed
  - Allemand : Kompakt
- Couleurs principales :
  - Primaire : #1E1E1E (gris foncé)
  - Secondaire : #FFFFFF (blanc)
  - Accent : #FF6F61 (rouge corail)
- Polices :
  - Titre : Roboto Bold, 16px
  - Texte : Roboto Regular, 14px
- Espacement :
  - Marges : 8px
  - Interlignes : 1.2
- Comportement :
  - Les éléments de l'interface sont plus rapprochés les uns des autres.
  - Les boutons et les champs de saisie sont légèrement plus petits pour économiser de l'espace.
  - Les icônes sont simplifiées pour une meilleure lisibilité dans un espace réduit.
- Disposition :
  - Le rack principal est caché en haut et en bas.
  - Un nouveau rack "de côté" est ajouté au dessus des éléments de côté - au dessus de la section des scores. Augmenter la largeur de ce rack pour qu'il soit plus visible et accessible, et que surtout toutes les tuiles tiennent en une seule ligne.
- Autres modifications de style pour le mode compact :
  - Augmentation de la taille des lettres dans les tuiles pour une meilleure lisibilité.
  - Augmentation de la taille des points dans les tuiles pour une meilleure visibilité.
  - Eclairissement du fond des tuiles pour un meilleur contraste avec le texte.
