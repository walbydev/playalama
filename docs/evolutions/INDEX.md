## Conventions

- **Statut : Proposition** → idée documentée, pas encore planifiée.
- **Statut : Planifiée** → inscrite dans la roadmap, jalons définis.
- **Statut : En cours** → développement actif.
- **Statut : Livrée** → fusionnée dans `master`, référencer la PR/commit.
